# Drum Kit Web Application

## DESCRIPTION
The Drum Kit project is a fun and interactive web application that allows users to play virtual drums by clicking on corresponding buttons or using keyboard keys. This project is built using **HTML, CSS, and JavaScript**, providing an engaging way to create different drum sounds and beats.

---

## FEATURES

### Interactive Drumming
Users can simulate playing drums by clicking on the drum pads or pressing the corresponding keys on their keyboard.

### Multiple Drum Sounds
Each drum pad produces a unique sound, offering a variety of beats to experiment with.

### Keyboard Shortcuts
Users can use the keys **"w", "a", "s", "d", "j", "k", "l"** to trigger the drum sounds easily.

### Visual Feedback
Each drum pad animates when clicked or when its corresponding keyboard key is pressed, creating a dynamic visual experience.

---

## HOW TO PLAY

### Play by Clicking
Click on any drum pad labeled **"w", "a", "s", "d", "j", "k", "l"** to produce the corresponding sound.

### Play by Keyboard
Press the keys **w, a, s, d, j, k, l** on your keyboard to play the same drum sounds.

### Create Beats
Mix and match different drum sounds to create your own rhythms and beats!

---

## TECHNOLOGIES USED

- **HTML** – Structures the layout and drum elements  
- **CSS** – Styles the interface, animations, and visual design  
- **JavaScript** – Handles user interactions, plays audio, and adds animations
